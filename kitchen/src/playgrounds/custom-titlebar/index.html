Custom Titlebar Demo

Custom Titlebar Implementation

This window uses titleBarStyle: "hidden" to hide both the native titlebar and window controls. The custom titlebar above is a regular HTML element with the .electrobun-webkit-app-region-drag class to make it draggable.

Window Controls

The close, minimize, and maximize buttons are custom HTML elements styled like macOS window controls. They use .electrobun-webkit-app-region-no-drag to remain clickable within the draggable region.

Test Checklist

  • Drag the window by the titlebar area
  • Click the red button to close
  • Click the yellow button to minimize
  • Click the green button to maximize/restore
