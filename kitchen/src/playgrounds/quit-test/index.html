These require manual action. The beforeQuit handler should still fire and log below.

Dock Icon Quit (macOS): Right-click the app icon in the Dock and choose "Quit"

SIGINT (Ctrl+C): If running from terminal, press Ctrl+C
SIGTERM: Send SIGTERM from another terminal: kill <pid>
