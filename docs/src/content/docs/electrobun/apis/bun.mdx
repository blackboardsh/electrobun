---
title: "Main Process API"
---
              
              <p>The main process API manages your application's lifecycle, creates windows, handles system events, and provides the bridge between your UI and the operating system.</p>

              <p>Electrobun is an npm dependency in your project. If you're just starting to look around, take a look at the Getting Started Guide first to learn how to set up your first project.</p>

<p>Electrobun bundles Cottontail to execute your TypeScript main process. Cottontail implements the Bun and Node.js compatibility APIs used by Electrobun applications.</p>

<p>Import the main process API from the source-compatible `electrobun/bun` entrypoint:</p>
                            
                
```javascript
import Electrobun from "electrobun/bun";

const win = new Electrobun.BrowserWindow(/*...*/);

// or

import {
  BrowserWindow,
  ApplicationMenu,
  // other specified imports
} from "electrobun/bun";

const win = new BrowserWindow(/*...*/);
```
