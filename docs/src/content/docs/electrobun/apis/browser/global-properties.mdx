---
title: "Global Properties"
---
              
              <p>These global properties are injected into the browser context regardless of whether you've instantiated the `Electroview`.</p>

              <h3>window.__electrobunWebviewId</h3>

              <p>The id of the webview</p>

              <h3>window.__electrobunWindowId</h3>

              <p>The id of the window</p>
