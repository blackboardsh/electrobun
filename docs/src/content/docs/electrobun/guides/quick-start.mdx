  <p>Welcome to Electrobun! This guide will help you create your first ultra-fast, tiny desktop application with TypeScript.</p>

## Prerequisites
  
  <p>Before getting started, make sure you have:</p>
  <ul>
    <li><a href="https://bun.sh" target="_blank">Bun</a> installed on your system</li>
    <li>A text editor or IDE (Blackboard's own <a href="https://blackboard.sh/colab/">co(lab)</a> recommended)</li>
    <li>Basic knowledge of TypeScript/JavaScript</li>
  </ul>

## Getting Started
  
  <p>Create a new Electrobun project with a single command:</p>
  
  
```
bunx electrobun init
```


  <p>It'll ask you which template project you want to get started with.</p>

  <p>This creates a new directory with the basic project structure:</p>

  
```
my-app/
├── src/
│   ├── bun/
│   │   └── index.ts        # Bun entry point (main process)
│   └── mainview/
│       ├── index.html      # UI template
│       ├── index.css       # Styles
│       └── index.ts        # Frontend logic
├── package.json            # Project dependencies
├── tsconfig.json
└── electrobun.config.ts    # Build configuration
```


## Running Your App

  <p>Navigate to your project directory and start development:</p>

  
```
cd my-app
bun install
bun start
```


  <p>This will use the Electrobun cli:</p>
  <ul>
    <li>Create a quick start project on your machine</li>
    <li>Use the Electrobun cli to do a dev build of your app</li>
    <li>Open your app in dev mode</li>    
  </ul>
