## Migration Notes

### CEF cache wipe on first launch after upgrade

              <p>First launch after upgrading silently wipes the CEF cache once. End users get logged out of any sites they were signed into via your app and client-side site data (cookies, localStorage, IndexedDB) is reset. WebView2 storage and your own app data are untouched.</p>

## New Features

              <ul>
                <li><strong>DirectComposition for transparent windows on Windows</strong>, with a zero-copy WGPU bridge that brings the bridged path from ~48 to ~1729 FPS.</li>
                <li><strong>File associations</strong> via a new <code>app.fileAssociations</code> config — opened files arrive through the existing <code>open-url</code> event.</li>
                <li><strong>Native <code>.icon</code> support on macOS</strong> — point <code>mac.icons</code> at an Icon Composer <code>.icon</code> file to ship Liquid Glass icons on macOS 26+ (with <code>.icns</code> fallback for older versions).</li>
                <li><strong><code>BrowserWindow.hide()</code></strong> with native bindings on all three platforms.</li>
                <li><strong>Linux desktop integration</strong> — apps drop a <code>.desktop</code> file into the XDG applications directory so they appear in the system app menu and launcher.</li>
                <li>Bumped Bun to 1.3.11 and CEF from 145 to 147.</li>
                <li>Surface full error output when <code>electrobun build</code> or <code>electrobun dev</code> fails.</li>
                <li>Renamed <code>Updater.localInfo.getLocallocalInfo()</code> → <code>getLocalInfo()</code> (old name kept as deprecation alias).</li>
                <li>Exported <code>RESOURCES_FOLDER</code> from <code>Electrobun.PATHS</code>.</li>
                <li>Moved Electrobun docs back into the main repo with a dedicated deploy workflow.</li>
              </ul>

## Bug Fixes

              <ul>
                <li>Fix <code>hiddenInset</code> frameless window on Windows — preserve resize borders and DWM shadow, and stop clipping content when maximized.</li>
                <li>Fix fullscreen behaviour and traffic light positioning on macOS.</li>
                <li>Use <code>xcrun</code> for macOS native builds.</li>
                <li>Fix Windows tar handling and resolve <code>rcedit</code> explicitly.</li>
                <li>Linux: persistent cookie storage now works under WebKitGTK; full-size BrowserViews render as a proper overlay instead of a 1×1 GtkFixed; CEF webviews stop opening in their own windows.</li>
                <li>Unify CEF partition path handling.</li>
                <li>Strip trailing slashes from <code>views://</code> URL paths.</li>
                <li>Allow <code>null</code> for <code>BrowserView.ptr</code> to avoid crashes on torn-down views.</li>
                <li>Include template <code>.gitignore</code> files when scaffolding new projects.</li>
                <li>Remove the spinner-thread deadlock in the Linux extractor and improve its progress UX.</li>
                <li>Remove a duplicate <code>makeSurfaceTexture</code> definition in the WebGPU layer.</li>
              </ul>
