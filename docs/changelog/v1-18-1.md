*Released 2026-05-04.*

## Bug Fixes

- Fix Windows updater applying updates to the wrong path on canary/beta channels.

- Fix Windows update cleanup race that could leave the new app installed inside the old app folder.

- Fix bspatch crashes on Windows after a recent Windows OS update (bumped bundled `zig-bsdiff`; `libsais` is now built with `zig cc` for a baseline-safe x86_64 ISA).
